# 事件

事件接口

  • $on(eventName) 监听事件
  • emit(eventName) 触发事件

vm.$emit(event, [...args])

  • event 派发事件名（String）
  • args 参数列表（Any）

vm.$on(event, callback)

通常业务中 emit 在父作用域进行触发，通知子组件进行执行事件
